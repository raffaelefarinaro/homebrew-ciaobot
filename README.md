# homebrew-ciaobot

Homebrew tap for [Ciaobot](https://github.com/raffaelefarinaro/ciaobot).

## Install

```bash
brew install raffaelefarinaro/ciaobot/ciaobot
```

Or tap once, then use the short name:

```bash
brew tap raffaelefarinaro/ciaobot
brew install ciaobot
```

## Upgrade

```bash
brew upgrade ciaobot
```

In-app updates from the Ciaobot Settings page also use `brew upgrade ciaobot` when the app was installed through this tap.

## Formula maintenance

The formula is updated automatically when a new [ciaobot release](https://github.com/raffaelefarinaro/ciaobot/releases) is published. See `scripts/update-homebrew-tap.sh` in the main app repository.
